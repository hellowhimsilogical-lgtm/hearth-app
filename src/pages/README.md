# Hearth

A cozy household organizer PWA with real-time Supabase sync.

## Features

- ✨ Magic link authentication
- 🔄 IndexedDB-first with Supabase backup
- 📱 Progressive Web App (works offline)
- 🎨 Accessible, neurodivergent-friendly design

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:5173

## Built with

- React + Vite
- Supabase
- TypeScript
